    Calculate source match rates

    Input

    To calculate source match rates, users should provide a VCF file containing phased genotypes from the reference, target, and source populations (e.g. test.match.rate.data.vcf). Users also need to provide three files containing names of individuals from the reference, target and source populations (e.g. ref.ind.list, tgt.ind.list and nean.ind.list) for analysis. The file (e.g. test.match.rate.score.exp.results.tsv) containing S* scores from sstar score is also required.

    Users can calculate source match rates with the following command:

    sstar matchrate --vcf tests/data/test.match.rate.data.vcf \
                --ref examples/data/ind_list/ref.ind.list \
                --tgt examples/data/ind_list/tgt.ind.list \
                --src examples/data/ind_list/nean.ind.list \
                --score tests/results/test.match.rate.score.exp.results.tsv \
                --output test.match.rate.results.tsv

    The expected result above can be found in test.match.rate.exp.results.tsv.

    Output

    An example of the output is below:

    chrom start end sample match_rate src_sample
    21 9400000 9450000 NA06986 0.0454545 Nean

    The meanings of the first five columns are the same as those in the output from sstar score. The meanings of the remaining columns:

    • The match_rate column is the source match rate on the current region. If the --phased argument is used, it reports the match rate for each haplotype; otherwise, it reports the average match rate across the two haplotypes carried by a diploid individual.
    • The src_sample column is the name of the individual from the source population used for calculating the source match rate.

    Settings

    Argument Description
    --vcf Path to the VCF file containing genotype data.
    --ref Path to the file containing reference individual IDs.
    --tgt Path to the file containing target individual IDs.
    --src Path to the file containing source individual IDs.
    --anc-allele Path to the BED format file containing ancestral allele information.
    --output Path to the output file.
    --thread Number of threads for multiprocessing. Default: 1.
    --mapped-region Path to the BED file containing mapped regions. If not provided, the full window is treated as mapped.
    --score Path to the score file generated by sstar score.
    --phased Report match rates for each haplotype.
